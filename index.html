Projects

Multi-Document AI Research Assistant (RAG)

Built a Generative AI application that enables users to upload multiple PDF documents and ask questions based on their content using a Retrieval-Augmented Generation pipeline.

Implemented document processing, chunking, embeddings, and semantic retrieval using LangChain and ChromaDB. Integrated LLaMA 3 via Groq API for context-aware answer generation and added conversational memory for handling follow-up queries.

Technologies: Python, Streamlit, LangChain, Groq (LLaMA 3), HuggingFace, ChromaDB
